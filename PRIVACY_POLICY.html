Privacy Policy for Brand Element Extractor

Last updated: January 17, 2025

Overview

Brand Element Extractor is committed to protecting your privacy. This Chrome extension operates entirely within your browser and does not collect, store, or transmit any personal data or browsing information to external servers.

Data Collection

We do not collect any data. All brand element extraction and analysis happens locally in your browser. The extracted brand guidelines are saved directly to your computer's Downloads folder.

Permissions Used

Our extension requires the following permissions to function:

  • activeTab: To analyze the currently active webpage's design elements when you click the extension button
  • storage: To remember whether you've seen the welcome screen (stored locally in your browser)
  • downloads: To save the generated brand guideline HTML file to your Downloads folder
  • scripting: To inject the analysis script into web pages for extracting brand elements
  • notifications: To notify you when the brand guideline download is complete

Data Processing

All data processing occurs locally:

  • Color extraction from CSS styles
  • Font family and typography analysis
  • Logo and image extraction
  • Button and spacing measurements
  • AI prompt generation

No data leaves your browser during this process.

Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

Data Storage

The only data stored is:

  • A flag indicating whether you've seen the welcome screen (stored in Chrome's local storage)
  • The generated HTML file saved to your Downloads folder

Updates

This privacy policy may be updated to reflect changes in the extension's functionality. Any updates will be reflected in the "Last updated" date at the top of this policy.

Open Source

This extension's source code is available for review, ensuring complete transparency in how your data is handled.
